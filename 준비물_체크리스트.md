## 🏗 배포 아키텍처 (보유 인프라 기준)
- **GitHub** — 코드 저장·버전 관리
- **Cloudflare (내 도메인)** — 프론트 정적 호스팅(Cloudflare Pages) + 예쁜 URL
- **Render** — Solar API 프록시(백엔드) → **키를 서버에 숨김** (보안 핵심)
- **Supabase** — 데이터 저장

```
[환자 브라우저] → Cloudflare(내 도메인) 화면
       ↓ 대화 전송
[Render 백엔드] → Solar API 호출(키 숨김) → 추출 결과 반환
       ↓
[Supabase] 저장 → [의사 대시보드] 리포트
```

### ⚠️ Render 무료 티어 함정
- 안 쓰면 잠들어(cold start) **첫 요청 ~30초 지연**.
- 발표 대비: ① 발표 직전 미리 1회 요청해 깨우기 ② 목업 데모를 백업으로 준비.
