## 1. 첫 화면 / 온보딩 (작업순서 1~2)
- [ ] 최초 로그인 뒤 가입 동의 화면 1회 표시 ("저는 AI예요…위험 감지 시 선생님께 알림")
- [ ] 입력창 "무엇이든 이야기해 보세요" · 한 글자 입력 시 ↑ 전송 버튼 활성화
- [ ] (데스크탑 ≥900px) 사이드바 고정 노출 + 햄버거로 접기/펼치기 / (모바일) 오버레이 드로어
- [ ] 한글 입력 후 Enter 1회에 1번만 전송 (IME 이중 전송 없음)

## 2. 범용 대화 (작업순서 3)
- [ ] 일상 메시지("오늘 좀 피곤했어") → 자연스러운 따뜻한 답변
- [ ] 코딩/잡담 질문도 일반 챗봇처럼 답함 (정해진 답 아님 = Solar 실연결 확인)
- [ ] 백엔드 죽여도(또는 cold start) 5초 뒤 목업 답변으로 폴백 — 대화 안 끊김

## 3. 백그라운드 추출 (작업순서 4) — 화면엔 안 보임
- [ ] 추출 표시가 화면에 **안 나타나는지** 확인 (조용히 동작 = 정상)
- [ ] 다음 문장들을 차례로 입력(리포트 재료 쌓기):
  - [ ] "어제 4시간밖에 못 잤어" (수면)
  - [ ] "약은 챙겨 먹었어" / 다른 턴에 "오늘 약 깜빡했어" (복약)
  - [ ] "회사 상사 때문에 너무 스트레스야" (스트레스원)

## 4. 위기 자원 카드 (작업순서 5) — 규칙 기반, 항상 동작
- [ ] "다 끝내고 싶어" 입력 → **따뜻한 공감 답변 + 위기 자원 카드**가 인라인으로 표시 (모달 아님, 대화 계속)
- [ ] 카드에 자살예방 109(24시간·무료) · 정신건강 위기상담 1577-0199 (탭하면 전화 연결)
- [ ] **백엔드 꺼도 카드가 뜨는지** 확인 (안전은 API 비의존 — 핵심 셀링포인트)

## 5. 리포트 + 환자 비공개 선택 (작업순서 6)
- [ ] 아바타(나) → "리포트 기록 보기" → 풀스크린 리포트
- [ ] 5영역 집계: ① 수면(평균+막대) ② 약물 순응도(복약 n/m일) ③ 안전 ④ 스트레스원 ⑤ 정신상태
- [ ] 위 4번을 했다면 맨 위 **⚠ 안전 트리아지 카드 + 근거 원문** 표시
- [ ] 항목 "공유 중 ↔ 비공개" 토글 동작 (비공개 시 흐려짐)
- [ ] **③ 안전은 🔒 항상 포함** — 토글 없음(빼기 불가) 확인
- [ ] "의사에게 보여주기" → 환자 통제 UI·비공개 항목 숨김, "‹ 환자 화면"으로 복귀
- [ ] "PDF 내보내기" → 인쇄 미리보기에 앱 메뉴·비공개 항목 빠지고 리포트만 나오는지

## 6. 설정 + 실연동 (작업순서 7)
- [ ] 아바타 → 설정 → 담당 병원·연락처 입력 후 **저장**
- [ ] 다시 위기 문장 입력 → 위기 카드에 **방금 입력한 병원 연락처가 표시**되는지
- [ ] 처방 약물 입력·저장 → 리포트 ② 약물 항목에 "처방: …" 표시
- [ ] 다음 진료일 입력 → 리포트 상단 "다음 진료" 표시
- [ ] 공유 기본값 토글 저장 → 리포트 새로 열 때 그 기본값으로 시작
- [ ] "가입 동의 고지 다시 보기" → 동의 안내 화면 다시 노출
- [ ] 새로고침 후에도 설정 유지 (localStorage)
